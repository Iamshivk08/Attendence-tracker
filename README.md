# 📌 Overview

The **Attendance Tracker System** is a web-based application designed to simplify attendance monitoring and provide a clear visualization of attendance records.

The system provides an interactive dashboard where users can view attendance statistics, track present and absent days, calculate attendance percentage, and analyze weekly attendance trends through graphical representation.

This project focuses on creating a modern, responsive, and user-friendly interface using frontend technologies. It demonstrates concepts such as DOM manipulation, dynamic data updates, animations, and data visualization.

---

# 🎯 Objectives

The main objectives of this project are:

- To create a simple and efficient attendance monitoring system.
- To visualize attendance data in an understandable format.
- To calculate attendance percentage automatically.
- To provide a dashboard-based user experience.
- To practice frontend development concepts.
- To implement interactive charts for better data analysis.

# 📊 Dashboard Components

## 🟢 Present Days

Displays the total number of days the student was present.

Example:

```
Present: 5 Days
```

---

## 🔴 Absent Days

Displays the total number of absent days.

Example:

```
Absent: 2 Days
```

---

## 📈 Attendance Percentage

Automatically calculates overall attendance percentage.

Formula:

```
Attendance Percentage =
(Present Days / Total Days) × 100
```

---

## 📅 Weekly Attendance Trend

Shows attendance data for the previous 7 days.

Example:

```
Monday     Present
Tuesday    Present
Wednesday  Absent
Thursday   Present
Friday     Present
Saturday   Absent
Sunday     Present
```
